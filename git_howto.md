# Подcказка по git
## Создание репозитория:
```sh
git init
```

## Добавление файла для отслеживания
```sh
git add
```

## Добваления коментария
```sh
git commit -m
```

## Записи изменений в коде
```sh
git log
```

## Краткий вывод истории записей
```sh
git log --oneline
```

## Переключение между различными ветками объекта
```sh
git checkout
```
